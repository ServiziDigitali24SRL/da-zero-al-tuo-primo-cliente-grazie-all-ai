# 🚀 Come partire in 3 passi (5 minuti)

> 🌍 **Altra lingua?** Il sistema lavora in qualsiasi lingua (inglese, русский, Deutsch, español…): fai gli stessi 3 passi e al Passo 3 incolla il messaggio nella tua lingua — trovi le versioni **English / Русский / Deutsch** pronte in cima al [README](README.md). Claude riconosce la lingua e conduce tutta la sessione in quella.

Non hai mai usato l'AI? Nessun problema. Segui questi 3 passi e sei operativo.

```
   PASSO 1              PASSO 2              PASSO 3
 ┌──────────┐        ┌──────────┐        ┌──────────────┐
 │ Scarica  │        │  Apri    │        │  Incolla il  │
 │  Claude  │  ───►  │  Cowork  │  ───►  │  messaggio   │
 │ Desktop  │        │          │        │  qui sotto   │
 └──────────┘        └──────────┘        └──────────────┘
   2 minuti            30 secondi          e sei partito!
```

## Passo 1 — Scarica Claude Desktop
Vai su **[claude.ai/download](https://claude.ai/download)**, scarica l'app per il tuo computer (Mac o Windows) e accedi. Va bene anche l'account gratuito per provare; per lavorare bene è consigliato il piano Pro.

## Passo 2 — Apri Cowork
Dentro Claude Desktop, apri **Cowork** (è la modalità che lavora con i file sul tuo computer). Crea o scegli una cartella dove vuoi tenere il tuo progetto, per esempio `marketing-mia-azienda`.

## Passo 3 — Incolla questo messaggio
Copia e incolla esattamente questo:

```
Clona la repo https://github.com/ServiziDigitali24SRL/da-zero-al-tuo-primo-cliente-grazie-all-ai
e segui le istruzioni del file CLAUDE.md: parti subito con la prima domanda.
```

Premi invio. Claude scarica il sistema e ti fa la prima domanda. **Rispondi con parole tue** — se non sai una risposta, te ne propone lui. Fatto: sei partito. 🎉

*(Usi Claude Code invece di Cowork? Stesso identico messaggio: funziona uguale e salva i file nella tua cartella.)*

**Preferisci la chat semplice su claude.ai?** Scarica il file `zero-to-market.skill` dalla repo, caricalo in Impostazioni → Skills, poi scrivi *"creami il marketing della mia azienda"*. *(Nota: nella chat web i materiali li vedi e li scarichi a mano; per averli salvati automaticamente in una cartella sul tuo computer usa **Cowork** o **Claude Code**.)*
