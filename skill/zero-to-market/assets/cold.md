# TEMPLATE — Cold Outreach

## Cold Email (4 tocchi)
1. **Aggancio:** {personalizzazione} + problema + meccanismo in 1 riga + CTA soft ("vale 15 minuti?") — ≤120 parole
2. **Bump:** 2 righe + micro caso studio
3. **Valore:** risorsa gratuita (lead magnet)
4. **Breakup:** "chiudo il file?" 
Oggetti: 2 varianti A/B ciascuna.

## DM (LinkedIn/IG, 3 tocchi)
connessione senza pitch → valore → invito soft

## Script Cold Call
Pattern interrupt 15s → permesso → problema/aggancio → 2 domande di qualifica → obiettivo: fissare call/demo → gestione 5 rifiuti tipici ("mandami una mail", "non ho tempo", "abbiamo già un fornitore", "non ci interessa", "quanto costa?")
